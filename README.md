# 🪨 Nutzung mit Obsidian 
Ich strukturiere und nutze diese Notizen selbst in **Obsidian**. Da alle Dateien hier in reinem Markdown (`.md`) geschrieben sind, eignet sich das Projekt perfekt, um es als lokalen "Vault" (Notizbuch) in Obsidian einzubinden! 

**So richtest du es dir ein:** 
1. Lade dir Obsidian kostenlos [hier herunter](https://obsidian.md/).
2. Navigiere in den Ordner wo du diese Projekt speichern möchtest.
3. Öffne dein Terminal in diesem Ordner und lade dir dieses Projekt auf deinen Rechner herunter: 
```bash
git clone https://github.com/ProgrammingAtTheCore/Rechnernetze
```
4. Öffne Obsidian und wähle dort deinen gewählten Ordner/Rechnernetze aus.
